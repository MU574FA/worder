    1. Starts with: gets you all words that starts with the supplied text
      E.G. 'CA' => CAT, CAR, CAP, CAN, etc...
    2. X-Starts: gets you all words that not starting with the supplied text
      E.G. 'CA' => all but words that starts with 'CA'
    3. Ends with: gets you all words that ends with the supplied text
      E.G. 'AT' => RAT, FAT, CAT, HAT, etc...
    4. X-Ends: gets you all words that not ending with the supplied text
      E.G. 'AT' => all but words that ends with 'AT'
    5. Includes: gets you all words that includes the supplied LETTERS
      E.G. 'XD' => all words that has the letters 'X' & 'D'
    6. Excludes: gets you all words that excludes the supplied LETTERS
      E.G. 'XD' => all words that does not has the letters 'X' & 'D'
    7. Wildcard: gets you all words that matchs the given wildcard
      E.G. 'C?T' => CUT, CAT, etc... *Note, length of the wildcard to be considered.
    8. X-Wildcard: gets you all words BUT that matchs the given wildcard
      E.G. 'C?T' => all words that does not starts with 'C' and ends with 'T'. *Note, length of the wildcard to be considered.
    9. Note: The result coming back from the server gets a maximum of 99-100 words only
